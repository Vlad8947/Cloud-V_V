Проект задумывался для реализации приёма/передачи фалов от клиента на сервер через приложение.

На данный момент реализовано:
	- GUI для авторизации/регистрации клиента
	- GUI рабочего пространства клиента
	- функционал авторизации/регистрации
	- хранение логина/пароля в базе данных сервера
	- выбор и передача нескольких файлов на сервер

Планируется реализовать:
	- GUI сервера
	- просмотр хранимых файлов клиентом на сервере
	- передача файлов клиенту из сервера
	- вывод на экран статистики передающихся файлов (полоса загрузки, проценты)


Графическая составляющая реализована на JavaFX с использованием SceneBuilder и реализацией функционала
окон в классах-контроллерах.
В качестве сборщика пакетов используется "Maven".
